There are tasks for three persons:
1. The issue owner is the one assigned to the corresponding issue
2. The reviewer is someone independent from the issue, chosen by the issue owner.

# Issue Owner Tasks
* [ ] Remove "WIP" flag, once finished.
* [ ] Assign reviewer (not involved in solving the issue)

# Reviewer Tasks
* [ ] Is the code clean?
* [ ] Is the code well-documented?
* [ ] Are all API fully specified?
* [ ] Are there test cases for all functions?
* [ ] Document your investigation results!
* [ ] If you agree, click on thumbs up
* [ ] Select to remove the feature branch upon merging
* [ ] Accept merge request
